Description
============================

Yahoo! OAuth Android example using signpost


Requirement
============================

Download signpost library from

  * http://code.google.com/p/oauth-signpost/

Need 2 .jar files

  * signpost-core-1.2.1.2.jar 
  * signpost-commonshttp4-1.2.1.2.jar 


Installation
============================

  * copy jar files to libs/
